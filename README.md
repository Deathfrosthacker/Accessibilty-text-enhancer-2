# A11yText Enhancer

A free, privacy-focused accessibility tool that makes text more readable for people with dyslexia, ADHD, low vision, and other reading differences.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Accessibility](https://img.shields.io/badge/WCAG-2.1%20AA-green.svg)

## Features

- **Dyslexia-Friendly Fonts** — OpenDyslexic, Lexend, Verdana, Comic Sans, and more
- **Bionic Reading** — Highlights word stems to guide eye movement
- **Full Typography Control** — Font size, line height, letter spacing, word spacing
- **Contrast Modes** — High contrast, dark mode, sepia, blue light filter, custom colors
- **Text-to-Speech** — Built-in speech synthesis with speed control
- **Line Focus Guide** — Dims non-hovered paragraphs to reduce visual clutter
- **Readability Stats** — Word count, reading time, grade level estimation
- **Export** — Save as HTML or plain text
- **Privacy First** — All processing happens in your browser. No data leaves your device.

## Deploy Your Own (Free)

### Option 1: GitHub Pages (Easiest)
1. Fork this repository
2. Go to **Settings → Pages**
3. Select **Deploy from a branch** → `main` → `/ (root)`
4. Your site will be live at `https://yourusername.github.io/a11ytext-enhancer`

### Option 2: Vercel (Recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repo
3. Deploy instantly with auto-updates on every push

## Usage

1. Paste or type text into the input area
2. Adjust controls to your preference
3. Export or print when ready

Your settings are automatically saved to your browser.

## Accessibility Standards

This tool follows WCAG 2.1 Level AA guidelines:
- Minimum 4.5:1 contrast ratios
- Keyboard navigable
- Screen reader compatible
- Resizable text up to 200%
